# Çıtkırıldım Fındık ve Fındık Ürünleri - Sipariş Formu

Giresunnuts sipariş formu uygulaması. Müşteriler ürün seçebilir, bilgilerini girebilir ve PDF sipariş formu oluşturabilir.

## Özellikler

- Ürün seçimi ve miktar belirleme
- Müşteri bilgileri girişi
- PDF sipariş formu oluşturma (Türkçe karakter desteği ile)
- Responsive tasarım

## Teknolojiler

- HTML5
- CSS3
- JavaScript (Vanilla)
- PDFMake (PDF oluşturma)

## Kullanım

1. `index.html` dosyasını bir web tarayıcısında açın
2. Ürünleri seçin ve miktarlarını belirleyin
3. "Bilgileri Gir ve PDF Oluşturmaya Geç" butonuna tıklayın
4. Müşteri bilgilerini doldurun
5. PDF'i indirin

## GitHub Pages ile Deploy

Bu proje GitHub Pages üzerinden yayınlanabilir:

1. Bu repository'yi GitHub'a push edin
2. Repository Settings > Pages bölümüne gidin
3. Source olarak "main" branch'ini seçin
4. `/ (root)` klasörünü seçin
5. Save butonuna tıklayın

Birkaç dakika içinde siteniz `https://[kullanici-adi].github.io/[repo-adi]` adresinde yayınlanacaktır.
